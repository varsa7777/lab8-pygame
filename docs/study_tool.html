🎮 Moving Squares Project

Project Mastery Study Tool — Master Pygame, Physics Simulation & Game Design Patterns

📚 Study Summary

This project demonstrates core concepts in game development, physics simulation, and event-driven programming.

Technical Themes Covered:

  • Game Loop Architecture: Initialization → Event Handling → Update → Render
  • Pygame API: Display, Events, and Drawing
  • Object-Oriented Data Design: Using `@dataclass` for structured data
  • Type Hints & Annotations: Improving code clarity and IDE support
  • Physics Simulation: Velocity, Position Updates, and Bouncing Collisions
  • Neighbor-Finding Algorithm: Distance calculations with `math.hypot()`
  • Behavioral AI: Fleeing from larger squares when too close
  • State Mutation & Brownian Motion: Random jitter and deterministic updates
  • Boundary Conditions: Wall collision detection and elastic reflection
  • Algorithm Complexity: O(n²) neighbor search in each frame
